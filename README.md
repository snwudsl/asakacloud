## Các dòng hosting chính

Asaka Cloud có 4 dòng hosting Minecraft chính, mỗi dòng phục vụ một nhu cầu khác nhau:

### 1. Basic Minecraft Hosting
Dành cho server nhỏ, test cấu hình, hoặc bạn mới muốn thử. CPU Intel Xeon hiệu năng cao, NVMe 10Gbps, bandwidth không giới hạn, đặt tại datacenter VNPT HCM.

### 2. Performance Minecraft Hosting
Dành cho server cần hiệu năng tốt hơn, phục vụ nhóm người chơi đông hơn. RAM DDR4 ECC, SSD NVMe U2 — theo Asaka Cloud, đây là dòng tối ưu cho server vanilla và các bản modpack nhẹ.

### 3. Folia Minecraft Hosting
Tối ưu riêng cho Folia — một fork của Paper chuyên xử lý đa luồng. Nếu server bạn có nhiều người chơi đồng thời và cần tận dụng đa lõi CPU, dòng này phù hợp hơn.

### 4. Ryzen Minecraft Hosting
Dòng cao cấp nhất. CPU AMD Ryzen 9 9900X/9950X — xung nhịp cao, xử lý mạnh, phù hợp với server modpack nặng (RLCraft, ATM 10, Pixelmon...) hoặc các network server quy mô lớn.
